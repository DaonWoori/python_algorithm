# 재귀(recursion)

### 재귀함수란?
> 하나의 함수에서 자신을 다시 호출하여 작업을 수행하는 방식으로 주어진 문제를 푸는 방법이다.

### 재귀 함수의 속성
1. 같은 형태의 더 작은 문제로 나누어 풀 수 있어야 한다. -> 부분 문제(Subproblem)
```
5! = 5 x 4!
4! = 4 x 3!
3! = 3 x 2!
2! = 2 x 1!
1! = 1 x 0!
0! = 1
```

2. base case(더 이상 부분 문제가 없는 경우)와  recursive case를 모두 생각해야 한다.
```
n! (n 팩토리얼)

# base case
n = 0인 경우 -> n! = 1

# revursive case
n > 0인 경우 -> n! = (n-1)! x n

```
