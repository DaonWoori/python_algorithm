# 유클리드 호제법

### 약수
> 정수 a, b가 있고 a가 0이 아닐 때, a가 b의 약수라는 말은,
> 
> b = ak를 만족하는 정수 k가 존재한다는 의미이다.

### 유클리드 호제법: $O(logN)$
```
GCD(A, B) = GCD(B, A%B)
      if A%B = 0 -> GCD = B
          else GCD(B, A%B)
```

**증명**
> $A = aG$, $B = bG$ 일 때, G가 최대 공약수가 위해 a, b는 서로소
>
> 1. $A = q\cdot B + r$
> 2. $aG = q\cdot bG + r$
> 3. $r = (a-qb)G$
> 
>    => $(a-qb)$와 $b$가 서로소임을 증명하면 공식이 성립
> 
> if) $(a-qb)$와 $b$가 서로소가 아니라면
> 1. $(a-qb) = mp$, $b = np$
> 2. $(a-q\cdot np) = mp$
> 3. $a = (nq + m)p$, $b = np$인데, 앞에서 a와 b는 서로소이므로 성립하지 않음
