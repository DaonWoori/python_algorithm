# 베라의 패션

### 문제❓
```
베라는 상의 N 벌과 하의 N 벌이 있다. i 번째 상의와 i 번째 하의는 모두 색상 i를 가진다. N 개의 색상은 모두 서로 다르다.

상의와 하의가 서로 다른 색상인 조합은 총 몇 가지일까?
```

### 제한사항⚠️
* 시간 제한: 2 초
* 메모리 제한: 256 MB

### 입출력⌨️
* 입력은 아래와 같이 주어진다.
    ```
    N
    ```
* 상의와 하의가 서로 다른 색상인 조합의 가짓수를 출력한다.\
* 1 ≤ N ≤ 2017
* N은 정수이다.

#### 입출력 예시
```
# 예제 입력 1 
1
# 예제 출력 1 
0

# 예제 입력 2 
2
# 예제 출력 2 
2

# 예제 입력 3 
5
# 예제 출력 3 
20
```

### 내 코드🖥️
```
import sys

N = int(sys.stdin.readline())

print(N * (N-1))
```
