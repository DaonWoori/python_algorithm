# 합

### 문제❓
```
n이 주어졌을 때, 1부터 n까지 합을 구하는 프로그램을 작성하시오.
```

### 제한사항⚠️
* 시간 제한: 1초
* 메모리 제한: 128 MB

### 입출력⌨️
* 첫째 줄에 n (1 ≤ n ≤ 10,000)이 주어진다.
* 1부터 n까지 합을 출력한다.

#### 입출력 예시
```
# 예제 입력 1 
3
# 예제 출력 1 
6
```

### 내 코드🖥️
```
n = int(input())
sum = 0

for i in range(1, n+1):
  sum += i

print(sum)
```
