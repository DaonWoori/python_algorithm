# 소인수분해

### 문제❓
```
정수 N이 주어졌을 때, 소인수분해하는 프로그램을 작성하시오.
```

### 제한사항⚠️
* 시간 제한: 1 초
* 메모리 제한: 256 MB


### 입출력⌨️
* 첫째 줄에 정수 N (1 ≤ N ≤ 10,000,000)이 주어진다.
* N의 소인수분해 결과를 한 줄에 하나씩 오름차순으로 출력한다. N이 1인 경우 아무것도 출력하지 않는다.

#### 입출력 예시
```
# 입력 예시1
72
# 출력 예시1
2
2
2
3
3

# 입력 예시2
9991
# 출력 예시2
97
103
```

### 내 코드🖥️
```
N = int(input())

num = N
i = 2

while num != 1:
  if num % i == 0:
    print(i)
    num = num // i
  else:
    i += 1
```
